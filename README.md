# Inversor_v0.0
Inversor monofásico de 500W 
## Sin revisión 
Primera versión, se desarrolló este inversor de 500W, con parametros de 12VDC para la entrada aprovechando la topología de push pull, en la salida se obtienen 220VAC 60Hz. 
Aún pueden haber algunos errores de diseño/cálculos ya que esta es una version inicial de prototipado y no una versión comercial.
