# Inversor_v0.0: Prototipo Monofásico de 500W

Este repositorio contiene la primera versión de un **inversor monofásico de 500W** diseñado para convertir corriente continua (DC) en corriente alterna (AC).

---

## ⚙️ Características Técnicas del Prototipo

* **Potencia Nominal:** 500W
* **Topología:** Se utiliza una configuración **Push-Pull** en la etapa de potencia, ideal para sistemas de baja tensión de entrada y con el objetivo de simplificar el circuito de control.
* **Tensión de Entrada:** 12 VDC
* **Tensión y Frecuencia de Salida:** 220 VAC a 60 Hz
* **Aplicación principal:** Uso experimental y como base para sistemas de baja potencia o **alimentación de emergencia/portátil**.

---

## Estado del Proyecto (Sin Revisión)

Esta es una **versión inicial de prototipado**. El diseño y los cálculos aún están **sujetos a posibles errores o ineficiencias**.

* **Advertencia:** No es una versión comercial final. Se recomienda precaución al replicar o usar el diseño, y es esencial realizar una **revisión exhaustiva de los cálculos** (principalmente en el dimensionamiento del transformador y los componentes de conmutación) y las pruebas de seguridad antes de su implementación.
